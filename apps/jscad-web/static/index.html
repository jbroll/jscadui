JSCAD

JSCAD is a tool for creating parametric 2D and 3D CAD designs using JavaScript code.

Code CAD offers more precise control over designs, enables parameterized customization of models, and integrates seamlessly with the web. With an open-source, functional API, complex models can be created with just a few lines of code.

Create designs in the editor, or drag and drop local files.
